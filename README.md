##  Internship Tasks Overview

### 1. Iris Flower Classification (Task 1)
**Objective:** To develop a model that can automatically categorize Iris flowers into three species: *Setosa, Versicolor, and Virginica*.
- **Details:** Performed extensive EDA using Pairplots and Heatmaps to identify feature importance. Built a classification model achieving high precision.
- **Technologies:** Python, Scikit-learn, Logistic Regression, Seaborn.
- **Result:** Achieved an **Accuracy of 96.67%**.
- [ Explore Task 1 Folder](./Task1_Iris_Classification)

### 2. Unemployment Analysis in India (Task 2)
**Objective:** To analyze the socio-economic impact of the COVID-19 pandemic on the Indian job market.
- **Details:** Processed large datasets to track unemployment trends across different states. Visualized the drastic transition before and after the 2020 lockdown.
- **Key Findings:** Identified a national unemployment peak of **76.74%** and recognized **Tripura** as a majorly affected region.
- **Technologies:** Pandas, Matplotlib, Plotly, NumPy.
- [ Explore Task 2 Folder](./Task2_Unemployment_Analysis)

###  3. Car Price Prediction (Task 3 - In Progress)
**Objective:** To build a predictive system that estimates the market value of used cars based on several input features.
- **Details:** Features like fuel type, transmission, mileage, and age are used to predict the selling price using Regression techniques.
- **Technologies:** Machine Learning, Regression Algorithms, Feature Engineering.
- [ Explore Task 3 Folder](./Task3_Car_Price_Prediction)
